sequenceDiagram
participant Client
participant Gateway
participant AuthService
participant Redis
participant SystemService

    Client->>Gateway: 请求 (带 JWT Token)
    Gateway->>Gateway: AuthGlobalFilter 校验
    Gateway->>Redis: 检查 Token 黑名单
    Redis-->>Gateway: 返回检查结果
    alt Token 在黑名单
        Gateway-->>Client: 401 未授权
    else Token 有效
        Gateway->>Gateway: 解析 Token，注入用户头
        Gateway->>SystemService: 转发请求
        SystemService-->>Gateway: 返回响应
        Gateway-->>Client: 返回响应
    end
    
    Note over Client,SystemService: 登录流程
    Client->>Gateway: POST /auth/login
    Gateway->>AuthService: 转发请求
    AuthService->>SystemService: 校验用户凭证
    SystemService-->>AuthService: 返回用户信息
    AuthService->>Redis: 创建 Refresh Token 会话
    AuthService-->>Gateway: 返回 JWT + Refresh Token
    Gateway-->>Client: 返回登录响应
